* 逆の発想のFusionCalc2というiPhoneアプリを見てインスパイアされた

* Kotlin使わないと忘れちゃいそう

そのくせ、きっちり広告を入れたり、課金処理入れたりして、せこいまねもしっかりしていたりします。

修正BSDライセンスなので、勝手にビルドしたり配布したりすることができますが、ビルドする際には、公開していない広告IDなどの文字列リソースを追加するか、該当部分を外す必要があります。
